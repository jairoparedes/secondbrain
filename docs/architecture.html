Flujo de un request típico — El navegador hace GET / o POST /api/... a nginx (puerto 80). Nginx enruta: si es /api/* pasa por FastCGI a Laravel (PHP-FPM), que valida el Bearer token con Sanctum, consulta Postgres y/o cachea en Redis, y responde JSON. Si es cualquier otra cosa, proxea al contenedor de Next.js. Trabajos pesados (embeddings, webhooks, mailing en fases futuras) se encolan en Redis y los consume el Queue Worker de forma asíncrona.
